
### Q06.08

Write a program to read a matrix (maximum size 10 Ã— 15) from the user and display the subscripts of the maximum element in the matrix. Your program should consist of two procedures: main is responsible for reading the input matrix and for displaying the position of the maximum element. Another procedure `mat_max` is responsible for finding the position of the maximum element. Parameter passing should be done via the stack. For example, if the input matrix is

$\begin{bmatrix}12 & 34 & 56 & 78\\23 & 45 & 67 & 89\\34 & 56 & 78 & 90\\45 & 67 & 89 & 10\end{bmatrix}$, 

the output of the program should be

    The maximum element is at (2,3),

which points to the largest value (90 in our example).
